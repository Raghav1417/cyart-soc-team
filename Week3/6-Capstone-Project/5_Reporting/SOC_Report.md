# SOC Incident Report

## Executive Summary
A simulated attack was conducted to evaluate detection and response capabilities.

## Timeline
- Nmap scan performed
- Metasploit exploit executed
- Wazuh detected suspicious activity
- Alerts analyzed
- IP blocked using CrowdSec

## Analysis
The logs showed SSH access and privilege escalation attempts mapped to MITRE techniques.

## Recommendations
- Deploy agents on all endpoints
- Enable real-time monitoring
- Apply security patches
- Improve access controls

## Conclusion
The SOC workflow successfully detected and mitigated the attack.
