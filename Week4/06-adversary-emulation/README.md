# SOC Adversary Emulation using MITRE Caldera & Wazuh

##  Objective
Simulate a spearphishing attack (MITRE ATT&CK T1566) using MITRE Caldera and evaluate detection capabilities in Wazuh SIEM.

##  Tools Used
- MITRE Caldera
- Wazuh SIEM
- Windows VM (Agent)
- Kali Linux (Attacker)

##  Steps Performed

1. Started Caldera server
2. Deployed agent on Windows VM
3. Executed T1566 (Phishing simulation)
4. Monitored logs in Wazuh dashboard
5. Created custom detection rule

##  Outcome
- Attack successfully simulated
- Wazuh detected suspicious activity
- Alerts generated for phishing indicators
