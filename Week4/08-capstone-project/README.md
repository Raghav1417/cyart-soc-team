# SOC Capstone Project - Incident Response

## Overview
This project demonstrates a complete SOC workflow including attack simulation, detection, triage, response, and reporting.

## Tools Used
- Metasploit
- MITRE Caldera
- Wazuh
- TheHive
- CrowdSec


## 1. Attack Simulation
A Samba vulnerability was exploited using Metasploit.

- Target: Metasploitable2
- Attacker: Kali Linux

## 2. Adversary Emulation
MITRE Caldera was used to simulate attacker behavior (T1210).


## 3. Detection
Attack was detected and mapped to MITRE ATT&CK (T1210).

## 4. Triage (TheHive)
A case was created and tasks were assigned.


## 5. Response (CrowdSec)
Attacker IP was blocked successfully.



## Conclusion
The incident was detected, analyzed, and contained successfully using SOC tools.
