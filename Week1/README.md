#  Log Analysis & SOC Monitoring Practice

##  Objective
This project demonstrates practical SOC (Security Operations Center) skills including:
- Log analysis using Windows Event Viewer
- Browser history forensic analysis
- Security event documentation
- Dashboard monitoring setup
- Alert rule configuration in SIEM tools



## Tools Used
- Windows Event Viewer
- wevtutil CLI
- Eric Zimmerman Tools (LECmd)
- LogParser Lizard
- Elastic SIEM / Kibana
- Wazuh
