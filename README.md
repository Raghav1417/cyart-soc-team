# cyart-soc-team

## SOC Alert-to-Response Project

This project demonstrates a complete Security Operations Center (SOC) workflow using tools like Wazuh, TheHive, and Google Sheets.

### Key Features:
- Alert classification with MITRE ATT&CK mapping
- Alert triage and threat validation
- Incident response documentation
- Evidence collection and preservation
- Full attack simulation and response
